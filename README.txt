Projeto Criado por Dyonata Eliezér Ambrósio Machado para a Disciplina de Fundamentos da Programação Web

O projeto consta da criação de sistema de empréstimos.
Existem dois tipos de usuários. O usuário Administrador tem permissão para fazer empréstimos e gerenciar
os usuários. Enquanto o usuário padrão pode apenas fazer empréstimos. 

Para acessar o sistema, rode o script .sql que está na pasta sql.
Esse script criará as tabelas do banco de dados e o primeiro usuário Administrador, 
cujo login é root@teste.com e a senha é 1234.

Deve-se iniciar a aplicação pelo arquivo login.php
